## 📌 Overview

**AI Symptom Checker** is a full-stack web application that bridges the gap between user symptoms and preliminary health insights using Machine Learning. Users describe their symptoms in natural language, and the AI model predicts potential conditions based on trained medical data.

This project demonstrates how AI can assist in **basic healthcare awareness**, offering a responsive, visually aesthetic interface and a robust backend API.

⚠️ **Medical Disclaimer:** This application is for **informational and educational purposes only**. It is NOT a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider for medical concerns.
